# Linux一键安装指令
```
git clone https://github.com/suminerProxy/suminerProxy.git && chmod -R a+x suminerProxy && cd suminerProxy && nohup ./start.sh &

```

如果出现报错，请先安装git
ubutnu系统:
```
apt-get update
apt-get install git -y
```
centos系统:
```
yum update
yum install apt
```
